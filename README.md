# 🚀 Linux Server Deployment Project

## 📌 Description
Déploiement et sécurisation d’un serveur Linux avec :
- Nginx Reverse Proxy
- Gunicorn
- SSH sécurisé
- UFW
- Fail2ban
- Netdata Monitoring

## 🧱 Architecture
Client → Nginx → Gunicorn → Python App

## 🔐 Security
- SSH hardening
- UFW firewall
- Fail2ban protection

## 📊 Monitoring
Netdata monitoring for:
- CPU
- RAM
- Network
- Disk
